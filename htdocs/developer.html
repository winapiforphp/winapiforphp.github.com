Getting started

  1. Pull the main project: 
    hg clone https://<youruser>@bitbucket.org/auroraeosrose/win-api-for-php .
    This will then pull the 3 sub-projects win-gdi, win-gui, and win-system.
  2. Next get the most recent binary-pack download from auroraeosrose's repo and unpack them to the directory you just cloned the repo to.
  3. Open a command-prompt: 
    start \path\to\win-api-for-php\projects\wingui.sln
    Visual Studio will start up with the win-gui project as it's default project. You could also double-click the project file in explorer.
  4. Make some changes.
  5. Commit and push changes to your repo.
  6. Request pull.
